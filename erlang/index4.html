说N遍Hello World，递归的使用

Hello World说一遍怎么够呢，说N遍吧。代码如下：

    -module(helloworld).
    -export([hw3/1]).

    % 打印N遍Hello world
    hw3(N) when N > 0 ->
        io:format("Hello World ~n"),
        hw3(N-1);

    hw3(0) -> ok.

这有什么好坑的呢：

    hw3(N) when N > 0 ->

when后面是一个断言(guard)，仅在它为true时函数会被执行。
